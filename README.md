# Equipamentos Checklist

This project uses Supabase as its backend. To run the app you must provide the connection credentials and admin password in an `.env` file.

## Required environment variables

- `VITE_SUPABASE_URL` – URL of your Supabase instance
- `VITE_SUPABASE_KEY` – public API key for the project
- `VITE_ADMIN_PASSWORD` – password required for admin actions

Copy `.env.example` to `.env` and fill in your credentials.
